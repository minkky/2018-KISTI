# 2018-KISTI 프로젝트 (Spatio-Temporal Join, 효율적인 시공간 조인 기법)
- 3차원 시공간(location<lat, lng>, time)을 index로 하는 table을 생성하고 range query가 주어지면 해당하는 데이터를 빠르게 접근할 수 있도록 함
